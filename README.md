SuperOffice

SuperOffice is a modern office space listing platform built with Next.js, React, and Tailwind CSS. It allows users to browse, explore, and book office spaces in major cities across Indonesia. The project features a clean UI, city and office detail pages, and a responsive design.

## Features

- Browse office spaces by city
- View detailed office information, features, and sales contacts
- Explore favorite cities and fresh office spaces
- Responsive and modern UI with Swiper integration
- Built with TypeScript and Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Folder Structure

- `src/app` — Next.js app directory (pages, layouts, routes)
- `src/components` — Shared UI components (Navbar, Header, BenefitSection)
- `src/features/cities` — City-related components, data, and types
- `src/features/offices` — Office-related components, data, and types
- `public/assets/images` — Images and icons

## Tech Stack

- [Next.js 16](https://nextjs.org/)
- [React 19](https://react.dev/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
